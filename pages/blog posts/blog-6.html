Custom cursor - February 18, 2020.

AIMS & OBJECTIVES

The aim is to create a unique customised cursor which enables the portfolio to stand out, to further enhance user experience.
To illustrate a custom cursor pointer that will be unique and different compared to typical cursors on the websites by the help of JavaScript, as JavaScript has the capability to execute more functions, I will also be using CSS properties to create the shape for the cursor.

Definition

A cursor is an indicator to display the current location for user interaction on the display gadgets that usually respond to input and converts it to a pointing device which looks like a shape of an arrow.

As the default display of a mouse cursor is in a shape of an arrow, therefore I decided to create a unique experience for the users by making a customised cursor that the users will be able to notice as they roam through the portfolio. I originally wanted to create the cursor by just using CSS but according to (Cope, 2019). CSS currently only supports certain functionality for the cursor properties. Hence, then I decided to illustrate the custom cursor by the help of JavaScript, as JavaScript has the facility to execute more functions, I will also be using CSS properties to create the appearance for the cursor. Allowing me to fully express and create the technology that was originally intended
The appearance of the cursor will be converted to a circular shape by the help of CSS. The cursor should change shape as it hovers over a link with the help of JavaScript and CSS. the reason for doing this is because it will determine the users that it must be a significant piece of the website, which will determine that it’s a link.

As soon as the shape of the cursor changes the cursor; property called pointer which looks like a hand with an index finger will also be displayed in the middle of the custom cursor so it makes it obvious to the users that it’s clickable. This will be done as this conveys the default idea of certain functionality when typically clicking on a link on a browser.

I would commend myself to be a beginner at using JavaScript, as I believe I have a long way to go until I fully understand the programming language to be proficient enough to take on bigger projects which include JavaScript. A few months ago I created a hamburger menu from CSS and JavaScript. This shows that I am comfortable enough to understand some of the fundamentals, which mainly require JavaScript animation.

Tools and Technologies

The tools that I used to create this project was mainly JavaScript to make the cursor function properly. I used the array method, event listeners, query selector method and function method. I also used CSS properties to create the shape of the cursor and CSS animation and transition for subtle animation when the cursor changes shape.
